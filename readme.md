Descrição do Projeto
Este projeto tem como objetivo implementar quatro das cifras mais importantes da história da criptografia utilizando apenas JavaScript Vanilla.
- Funções JavaScript  
- for
- Arrays  
- Manipulação de strings e códigos ASCII  
- %  

As cifras implementadas preservam o **case** (maiúsculas/minúsculas) e mantêm **espaços, números e pontuações** inalterados.

Cifras Implementadas
1. **Cifra de César**  
   Uma cifra de substituição simples que desloca cada letra do alfabeto por um número fixo de posições.

2. **Cifra de Vigenère**  
   Uma cifra polialfabética que utiliza uma palavra-chave para determinar o deslocamento de cada letra.

3. **Cifra Atbash**  
   Uma cifra monoalfabética que substitui cada letra pela sua correspondente invertida no alfabeto (A ↔ Z, B ↔ Y, etc.).

4. **Cifra de ROT13**  
   Uma cifra de substituição que desloca cada letra 13 posições no alfabeto, sendo uma variante simples da Cifra de César.
